# Scapy-OpenCV

分析PCAP封包，從PCAP封包中取得HTTP裡面的 圖片資料，並將這些圖片資料進行人工智慧比對，快速篩選

* faces                              有辨識出人臉的圖片資料夾
* pictures                           從pcap中抓出的圖片
* bhp.pcap                           圖片被cache的pcap
* bhp2.pcap                          成功抓到圖片的pcap
* haarcascade_frontalface_alt.xml    OpenCV用
* pic_carver.py                      主要程式
